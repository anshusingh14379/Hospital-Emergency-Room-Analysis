# Hospital Emergency Room – Excel Dynamic Dashboard

### Project Overview
- This project focuses on analyzing Hospital Emergency Room (ER) data using Microsoft Excel to create a dynamic and interactive dashboard.
- The objective is to transform raw healthcare data into meaningful insights that help understand patient flow, waiting times, satisfaction levels, and operational efficiency.
- The dashboard is designed for decision-makers to quickly monitor key performance indicators (KPIs) and explore trends using interactive filters.

### Objectives
- Analyze patient inflow and ER performance metrics
- Monitor average waiting time and service efficiency
- Track patient satisfaction trends
- Understand admission patterns and referral departments
- Build an interactive, user-friendly Excel dashboard

### Tools & Technologies Used
#### Microsoft Excel
   - Power Query (Data Cleaning & Transformation)
   - Pivot Tables & Pivot Charts
   - Calculated Fields & KPIs
   - Slicers for dynamic filtering
   - Dashboard design & formatting

### Key KPIs & Analysis
- Total number of patients (trend analysis)
- Average waiting time
- Patient satisfaction score
- Admission vs non-admission status
- Age group distribution
- Gender-wise patient analysis
- Department-wise referrals
- Patients attended within defined time limits

### Data Cleaning & Preparation
- Removed inconsistencies and handled missing values
- Standardized column formats and data types
- Created calculated columns for better analysis
- Ensured data quality using Power Query

### Dashboard Features
- Fully interactive dashboard with slicers
- Real-time filtering by date, age group, gender, and department
- Clean and intuitive visual layout
- KPI cards for quick performance tracking

### Key Learnings
- Working with real-world–style healthcare data
- Importance of data cleaning before analysis
- Building dynamic dashboards using Excel
- Creating meaningful KPIs for decision-making
- Improving data visualization and storytelling skills
